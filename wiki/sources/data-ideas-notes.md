# Source: Открытые датасеты по C-BARQ

**Исходный файл:** [`datatsets/_data_ideas.md`](../../datatsets/_data_ideas.md)

## Ключевые выводы

1. Полного открытого C-BARQ датасета **не существует** — доступ фрагментированный.
2. UPenn база (~50k собак) — research database, не open dataset.
3. Реально доступные open-ish источники:
   - PLOS ONE 2024 supplementary (~435 собак, CSV)
   - Dog Aging Project (заявка)
   - Morris Animal Foundation Data Commons (регистрация)
4. Исследователи используют 3 модели доступа: UPenn, локальные опросы, вторичные когорты.
5. Причины закрытости: приватность, лицензирование инструмента, научный контроль.
